# 编码缓存自动维护 - Crontab 配置示例
# 
# 使用方法：
# 1. 编辑 crontab: crontab -e
# 2. 添加以下任一配置行
# 3. 保存并退出
#
# 注意：请根据实际项目路径修改脚本路径

# 每天凌晨2点执行缓存维护
0 2 * * * /Users/wangdada/Downloads/mcp/excel-mcp/chatExcel-mcp-server/scripts/cache_maintenance.sh maintain

# 每周日凌晨3点执行缓存维护（推荐）
0 3 * * 0 /Users/wangdada/Downloads/mcp/excel-mcp/chatExcel-mcp-server/scripts/cache_maintenance.sh maintain

# 每天中午12点创建缓存备份
0 12 * * * /Users/wangdada/Downloads/mcp/excel-mcp/chatExcel-mcp-server/scripts/cache_maintenance.sh backup

# 每小时检查缓存状态（仅用于调试，生产环境不推荐）
# 0 * * * * /Users/wangdada/Downloads/mcp/excel-mcp/chatExcel-mcp-server/scripts/cache_maintenance.sh stats

# 时间格式说明：
# 分钟(0-59) 小时(0-23) 日期(1-31) 月份(1-12) 星期(0-7，0和7都表示星期日)
#
# 常用时间示例：
# 0 2 * * *     - 每天凌晨2点
# 0 */6 * * *   - 每6小时执行一次
# 0 3 * * 0     - 每周日凌晨3点
# 0 1 1 * *     - 每月1号凌晨1点
# 30 2 * * 1-5  - 周一到周五凌晨2点30分

# 安装说明：
# 1. 确保脚本有执行权限：
#    chmod +x /path/to/cache_maintenance.sh
#
# 2. 测试脚本是否正常工作：
#    /path/to/cache_maintenance.sh help
#
# 3. 编辑 crontab：
#    crontab -e
#
# 4. 添加上述配置行之一
#
# 5. 查看当前 crontab 配置：
#    crontab -l
#
# 6. 查看 cron 日志（macOS）：
#    tail -f /var/log/system.log | grep cron
#
# 7. 查看维护日志：
#    tail -f /path/to/project/cache_maintenance.log
